🔎 Comment effectuer une vérification ?

  1. Saisissez le numéro d’identification dans le champ prévu.
  2. Cliquez sur le bouton Vérifier.
  3. Le système analyse les informations disponibles.
  4. Le résultat de la vérification s’affiche à l’écran.

⏱️ Le traitement prend généralement quelques secondes.
